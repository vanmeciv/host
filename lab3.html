Below is a depiction of Washington State Fast Flowing Flood Hazard areas in relation to Mt. Rainier Glaciers. Additional, there is a data layer intended to display the lahar evacuation routes within Pierce County.

This is the origin of the added layer to depict Lahar Evacuation Routes based on Carbon and Puyallup River: https://gisdata-piercecowa.opendata.arcgis.com/datasets/lahar-evacuation-routes

This is the origin of the added layer to depict Deep or Fast Flowing Floodways in Pierce County on the south side of the Narrows Bridge: https://gisdata-piercecowa.opendata.arcgis.com/datasets/deep-and-or-fast-flowing-floodway In order to fit the file into a Github upload, it required removing all JSON data from the north side of the bridge and Lake Tapps.

This is the origin of the added layer to depict Mt. Rainier Nat'l Park with a marker: https://gisdata-piercecowa.opendata.arcgis.com/datasets/parks-polygons
